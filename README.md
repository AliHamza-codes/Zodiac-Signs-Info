# Zodiac Discovery System

This project is a simple Python script that determines your Zodiac sign based on your date of birth and provides detailed information about your sign.

## Features
- Input your date of birth (YYYY-MM-DD)
- Automatically detects your Zodiac sign
- Displays key traits, element, ruling planet, and lucky color for your sign

## Usage
1. (Optional) Create and activate a virtual environment:
   - Windows: `venv\Scripts\activate`
   - macOS/Linux: `source venv/bin/activate`
2. Run the script:
   - `python mynew2.py`
3. Enter your date of birth when prompted (format: YYYY-MM-DD)

## Example
```
Enter your Date of Birth (YYYY-MM-DD): 1995-05-20
--- Results for Taurus ---
**Element:** Earth
**Trait:** Reliable & Patient
**Planet:** Venus
**Lucky Color:** Green
```

## Requirements
- Python 3.x
- No external packages required
